List集合

一、介绍

List 是有序集合的顶层接口。

特点：

   1.元素有序

​   2.元素可通过索引遍历

​   3.可以重复

常用子类：

​   ArrayList

​   LinkedList

二、详解

1.ArrayList集合

​ （1）特点：

​    底层使用数组

​    增删慢，查找快

​ （2）常见问题

​     ArrayList是线程不安全的，效率高

​     ArrayList初试容量为10，超过初始化容量，将以原来的1.5倍增长。即：当第一次超出容量10时，自动扩容至15
