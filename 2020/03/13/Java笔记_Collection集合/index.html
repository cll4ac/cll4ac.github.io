#Collection集合
##一、概念

   java中的集合是指存放一组数据的容器。

##二、体系

   Collection集合是java单列集合中的顶层接口，其抽取所有集合的共性方法，所以一般使用集合的时候，我们都用多态。
   单列集合：集合中的元素是独立的。

ps：总结的是常用的集合类和接口

##三、详解

1.方法

方法名 作用
boolean add(E e) 添加元素
boolean isEmpty() 判断集合是否为空。为空，则返回true，反之返回false
void clear() 移除集合所有元素
boolean contains(Object o) 判断集合是否存在该元素
boolean remove(Object o) 移除指定元素
Iterator iterator() 返回一个迭代器
int size() 返回集合中元素的个数
Stream stream() 获取去一个流对象

Code

3.总结

Collection的方法都是单列集合的共性抽取的，所以每个集合都有其特点和特性，所以一般不会使用Collection集合。
