css

1、介绍

 全称叫Cascading Style Sheets,即层叠样式表。

2、作用

 用与解决内容与表现分离的问题，内容指标签，表现指标签内的属性，使得页面更加简洁，开发效率极大提高。

3、使用

有3中方式使用，并且范围越小的优先权会大于范围越大的优先权。

 1、行内样式

   1、格式：直接写在元素中

html

   2、特点：只针对对当前元素有效

 2、内部样式表

   1、格式：写在标签内部

html

   2、特点：只作用于当前的页面

 3、外部表样式

   1、格式：写在.css文件，在标签内引用

html

   2、特点：针对引用该文件的页面有效

 4、注意

  当相同的属性作用在同一个标签时，将优先使用权为：

       行内样式 > 内部样式 > 外部样式

  并且只覆盖重叠的属性。
